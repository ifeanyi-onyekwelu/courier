# DashDrop Courier Site

Welcome to the DashDrop Courier Site repository! This repository contains the source code and project files for the DashDrop courier website.

## Introduction

The DashDrop Courier Site is a web application developed to streamline and enhance the courier service experience. It provides a user-friendly interface for customers to schedule pickups, track shipments, and manage their deliveries with ease.

## Features

- Easy-to-use interface for customers to create and manage shipments.
- Seamless order tracking to monitor the status of shipments in real-time.
- Secure authentication system for customer accounts.
- Integrated payment options for convenient and secure transactions.
- Admin dashboard for managing shipments, customers, and delivery personnel.
- Automated notifications and email alerts to keep customers informed about their shipments.
- Responsive design to ensure a consistent experience across different devices.

## Technologies Used

- Django: A high-level Python web framework used for rapid development and clean design.
- HTML/CSS: Front-end markup and styling.
- JavaScript: Used for interactive features and enhancements.
- Bootstrap: A popular CSS framework for responsive and mobile-first design.
- PostgreSQL: A powerful, open-source relational database management system.
- SMTP Email Backend: Integrated email functionality for sending notifications to customers.
